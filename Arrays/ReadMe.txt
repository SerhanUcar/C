This program does those:
(a) Reads an array that given by user and 
(b) Prints N integer elements of this array. Then: 
(c) Prints all negative elements from the array, 
(d) Finds and prints the sum of all array elements,  
(e) Finds and prints the maximum and minimum element in the array, 
(f) Counts total number of even and odd elements in the array, 
(g) Reverses the order of elements in the array, 
(h) Puts even and odd elements of the array in two separate arrays, 
(i) Searchs a given element in the array, 
(j) Checks if all the elements in the array are sorted,
(k) Finds the element that is the most closest to the average element.
